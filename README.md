# QuizApp

QuizApp is a Python-based interactive quiz application designed to offer a fun and engaging way to test your knowledge. With a user-friendly interface and a competitive leaderboard feature, QuizApp is perfect for both individual and group play.

## Features

- **User-Friendly Interface:** The application starts by asking for the user's name to personalize the experience.
- **Interactive Quiz:** Users can answer a set of questions without the pressure of a timer.
- **Score Display:** At the end of the quiz, users can view their individual scores.
- **Continuous Accessibility:** The application remains active, allowing multiple users to participate in the quiz sequentially.
- **Leaderboard Feature:** A leaderboard displays the top performers, encouraging a competitive spirit. Users also have the option to clear the leaderboard as needed.

## How to Access

1. **Clone the Repository:**
   ```sh
   git clone https://github.com/harshith1729/QuizApp.git
   cd QuizApp
   ```

2. **Install the Necessary Libraries:**
   Ensure you have the following libraries installed:
   - `tkinter`: GUI creation.
   - `messagebox`: Dialog boxes.
   - `ttk`: Modern themed widgets.
   - `ttkbootstrap`: Advanced GUI styling.
   - `os`: File handling and OS interactions.

   You can install these libraries using pip:
   ```sh
   pip install tkinter ttkbootstrap
   ```

3. **Run the Application:**
   ```sh
   python main.py
   ```

## Libraries Used

## How to Use

1. **Launch the Application:** Run the `main.py` file.
2. **Enter Your Name:** The application will prompt you to enter your name for a personalized experience.
3. **Take the Quiz:** Answer the quiz questions displayed on the screen.
4. **View Your Score:** At the end of the quiz, your score will be displayed.
5. **Check the Leaderboard:** See the top performers and your ranking.
6. **Play Again:** The application allows multiple users to participate sequentially.
